# Tuborg API Backend Server

## Manual Setup

1. **Adjust .env file due to the example.env**

    ```bash
    MYSQL_USER="please enter mysql user"
    MYSQL_PASSWORD="please enter mysql password"
    MYSQL_DATABASE="please enter mysql database name"
    MYSQL_HOST="please enter mysql database host"
    TOKEN_KEY="please enter your token"
    GOOGLE_API_KEY="please enter your key"
    ```

2. **Install dependencies**

    ```bash
    npm install
    ```

3. **Run Server:**

    ```bash
    npm start
    ```


Than use http://localhost:3005/api for requests.
